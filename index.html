Passerine

A small extensible programming language designed for concise expression with little code.

Why Use Passerine?

Light and Fast

Passerine’s entire implementation, from lexer to VM, is only ~2000 lines of code. Clever memory management techniques and an efficient interpreter means the runtime is even smaller. Passerine’s package manager, Aspen, helps manage everything for you, so development’s a breeze.

The implementation ↗

Easily Embeddable

Passerine’s VM is written in Rust. A well-designed FFI allows each language to communicate with each other, allowing for rich usage of each language’s libraries. Passerine’s memory management technique, vaporization, also plays friendly with Rust’s borrow checker, allowing for easy integration.

Get Involved!

Install

Downloading Passerine is quick and easy. Just run the following in the shell of your choice:

sh <(curl -sSf https://passerine.io/install.sh)

This script just pulls the latest version of Passerine, builds it from source, and adds it to your $PATH. Give it a read before running it if you'd like!

In the future, we plan to distribute prebuilt binaries, but for now, both Git and Cargo are required.

To create and run a new Passerine project:

$ aspen new hello-passerine
$ cd hello-passerine
$ aspen run
